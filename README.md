# pages
Persion php + react web site

## dependence
- php7
- node

## configure
- php copy file `conf/config.sample.php` as `conf/config.php`
- js copy file `front/src/config/dev.js` as `front/src/config/local.js`

## dev
1. `port=5000 npm run dev`
2. open `localhost:5000`

## build
`npm run build`
